# Coast Partnerships Plugin

The partnerships team's toolkit for Coast. Download the .plugin file and install it to get all four workflows.

## What's Included

**Skills:**
- quarterly-partner-payout — Generate partner rev share payout spreadsheets
- - partner-brief — Produce a Partner Marketing Brief for new partners
  - - weekly-partnership-kpi — Generate the weekly Partnership KPI update
    - - market-landscape-report — Weekly market landscape briefing
     
      - **Slash Commands:**
      - - /quarterly-payout — Run the quarterly partner rev share payout
        - - /partner-brief [name] — Generate a marketing brief for a partner
          - - /weekly-kpi — Draft this week's KPI update
            - - /market-scan — Run the weekly market landscape report
             
              - ## How to Install
             
              - 1. Download coast-partnerships.plugin from this repo
                2. 2. Open Claude and install the plugin file
                   3. 3. Make sure you have Notion, Slack, Gmail, Calendar, and Granola connectors set up
